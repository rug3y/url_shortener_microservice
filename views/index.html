URL Shortening Microservice - FreeCodeCamp Project

User Stories:

  • User Story: I can pass a URL as a parameter and I will receive a shortened URL in the JSON response.
  • User Story: If I pass an invalid URL that doesn't follow the valid http://www.example.com format, the JSON response will contain an error instead.
  • User Story: When I visit that shortened URL, it will redirect me to my original link.

Example Usage:

To see a directory of saved urls:

https://url-shorten-r.herokuapp.com/urls

To create a new short url:

Input: https://url-shorten-r.herokuapp.com/new/http://example-url.com

Output: {"__v":0,"original":"http://example-url.com","short":"2232","_id":"573fd1a7c7ea7b691c59b171"}

To use the shortened url, pass the "short" url that was given:

https://url-shorten-r.herokuapp.com/urls/2232
